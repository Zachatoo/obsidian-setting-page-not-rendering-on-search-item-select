# Setting page not rendering on search item select

1. Clone this repo into a test vault (optional, can instead copy build files into test vault).
1. Build plugin (with `npm run build` or `npm run dev`).
1. Open settings.
1. Search for "page" and select search result in search.
    - Page will not render.
1. Search for "items" and selelct search result in search.
    - Page will render.
